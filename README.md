# Git and GitHub

- Step 1: **Creating a Local Repository**
```
    cd my_app
```
- Step 2: **Initialize a new Git repository**
```
    git init
```
- Step 3: **Adding files to the Git repository**
```
    git add filename
        or
    git add . 
    // the '.' refers to current directory
```
- Step 4: **Initialize a new Git repository**
```
    git init
```
